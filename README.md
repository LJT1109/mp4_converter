# MP4 to Pixel Mapping Converter

A web-based tool to convert MP4 videos or image sequences into pixel mapping data (.npy format) for DMX control or LED matrices.

## Features
- **Web Interface**: Easy upload and control via browser.
- **Preview**: Real-time grid visualization of the pixel data.
- **Conversion**: Converts MP4 or Image Sequences to `.npy` arrays.
- **Modes**: Supports RGB and Grayscale modes.
- **Playback**: Play, pause, and seek through the converted data.

## Usage
1. Install dependencies: `pip install -r requirements.txt`
2. Run the server: `python app.py`
3. Open browser at `http://localhost:5050`
